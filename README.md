# Супер Маріо 2D

🍄 Класична 2D гра в стилі Маріо

## Файли проекту

- **index.html** - HTML структура та CSS стилі
- **game.js** - Ігрова логіка на JavaScript
- **server.py** - Python HTTP сервер для запуску гри
- **mario.cpp** - C++ версія гри з ASCII графікою
- **compile_mario.bat** - Скрипт для компіляції C++ версії

## Як запустити гру

### Спосіб 1: Через Python сервер (рекомендовано)

```bash
python server.py
```

Або:

```bash
python3 server.py
```

Потім відкрийте у браузері: http://localhost:8000

### Спосіб 2: Через інший HTTP сервер

Використовуйте будь-який інший HTTP сервер (npx http-server, live-server, тощо) з директорії проекту.

### Спосіб 3: C++ версія (консольна)

Для запуску C++ версії потрібен компілятор g++:

```bash
# Компіляція
g++ mario.cpp -o mario.exe

# Запуск
mario.exe
```

Або використайте готовий скрипт:
```bash
compile_mario.bat
```

**Керування C++ версії:**
- **A/D** - Рух вліво/вправо
- **W або Space** - Стрибок
- **R** - Рестарт
- **Q** - Вихід

## Керування

- **← →** або **A D** - Рух вліво/вправо
- **↑** або **W** або **Space** - Стрибок
- **Shift** - Прискорення

## Особливості гри

- 10 рівнів з процедурною генерацією
- Система життів та очок
- Вороги: Goomba та Koopa
- Монети, блоки з питаннями, труби
- Камера слідкує за гравцем
- Колізія для всіх об'єктів

## Технології

- HTML5 Canvas для рендерингу
- JavaScript для ігрової логіки
- Python для HTTP сервера
- CSS для стилізації
